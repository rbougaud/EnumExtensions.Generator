EnumExtensions.Tool

Lightweight enum extension generator for .NET projects.
Features :
- Generates strongly-typed enum extensions
- Optimized ToStr, Parse, TryParse
- Case-insensitive parsing
- IsDefined, GetNames, GetValues
- Incremental generation
- CI friendly
- Supports nested enums (next version)
- Attribute-driven generation (next version)

************ Commands **********
dotnet tool install EnumExtensions.Tool --version 1.0.0
Command to execute the tool : dotnet tool run enumextensions
